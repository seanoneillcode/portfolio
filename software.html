Technologies and processes

I've been part of traditional, Scrum and Agile methodologies. I've used Java for 5 years, JavaScript for 3 years and have non-professional experience with Groovy, Scala, C, Python. I enjoy learning about esoteric languages and language design in general and have a strong preference for using functional languages. I've used associated technologies and standards such as Spring, Hibernate, ActiveMQ, Apache Servers, Jetty, SOAP, XML, JSON, Node, AngularJS, React, MongoDB, MySQL, CSS, HTML, JUnit, Mocha, Jasmine, Grunt, jQuery, Backbone. I enjoy pair programming, mob coding and think code review is vital. I think code should be easy to read, understand and modify.
