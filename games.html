Ultra Violet Gladiator

About

For Ludum Dare 38, the theme was 'one room'. I created a single player arena fighter where the player runs around hitting enemies to survive. It was created and polished across 4 days.

Development

I chose to use the LibGdx framework as it handles the rendering, playing sounds, collision handling and input handling that I needed. It's very portable and produced a releasable build using Gradle with a few commands.

What worked

  • bright colors
  • smooth animations
  • music and sound effects were good
What did not work
  • enemies chose targets poorly
  • players could win by running away
  • music was repetitive
  • controls were too awkward
For the release version I used the Ludum Dare feedback to improve the game. The game needed contrast between the fights so I inserted a night time break scene now and then where players can talk to other people and upgrade their stats. The upgrades require tokens which are dropped by dead enemies, forcing the players to engage rather than run away. I added a better song, fixed the controls and added proper menus. In the end however, I felt that the gameplay was fundamentaly too simple to be fun. It needs a more complex attack and defense system.
